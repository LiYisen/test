# 双均线交叉交易策略说明文档 (MA Strategy)

## 1. 策略概述

双均线交叉策略是一种经典的趋势跟踪策略。该策略通过计算短期和长期两条移动平均线，在短期均线上穿长期均线（金叉）时做多，在短期均线下穿长期均线（死叉）时做空。

### 1.1 核心理念
- **趋势识别**: 通过短期均线与长期均线的相对位置判断市场趋势方向
- **交叉信号**: 均线交叉点作为趋势反转的确认信号
- **延迟执行**: 信号在下一根K线开盘价执行，避免使用未来数据

### 1.2 策略特点
- 纯技术分析策略，逻辑简洁清晰
- 适用于日线级别的趋势行情
- 支持多空双向交易
- 信号延迟一天执行，严格遵循时序原则

### 1.3 策略参数

| 参数 | 类型 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| short_period | int | 5 | 2-50 | 短期移动平均线周期 |
| long_period | int | 20 | 5-200 | 长期移动平均线周期 |
| leverage | float | 1.0 | 0.1-10.0 | 账户承担的百分比风险系数 |

## 2. 均线计算

### 2.1 简单移动平均线（SMA）

策略使用收盘价计算简单移动平均线：

```
SMA(N) = (C1 + C2 + ... + CN) / N
```

其中 Ci 为第 i 根K线的收盘价，N 为周期。

### 2.2 均线状态管理

StateManager 维护以下状态：

| 字段 | 类型 | 说明 |
|------|------|------|
| prices | []float64 | 收盘价序列（保留最近 2*longPeriod 个） |
| shortMA | float64 | 当前短期均线值 |
| longMA | float64 | 当前长期均线值 |
| prevShortMA | float64 | 上一根K线的短期均线值 |
| prevLongMA | float64 | 上一根K线的长期均线值 |

### 2.3 就绪判断

当收盘价序列长度 >= longPeriod 时，策略进入就绪状态，开始产生信号。在此之前不产生任何交易信号。

### 2.4 内存管理

为避免价格序列无限增长，当价格数量超过 `2 * longPeriod` 时，只保留最近 `2 * longPeriod` 个价格数据。

## 3. 信号生成逻辑

### 3.1 金叉与死叉

**金叉（做多信号）**：
```
prevShortMA <= prevLongMA 且 shortMA > longMA
```
短期均线从下方穿越长期均线，表示上升趋势确立。

**死叉（做空信号）**：
```
prevShortMA >= prevLongMA 且 shortMA < longMA
```
短期均线从上方穿越长期均线，表示下降趋势确立。

### 3.2 信号延迟执行

交叉信号产生后，不会在当天立即执行，而是记录为待执行信号（PendingSignal），在下一根K线开盘时执行：

```
T日收盘后：
  检测到金叉/死叉
  记录 PendingSignal{Direction, Date=T日}

T+1日开盘：
  使用T+1日开盘价执行交易
  清除 PendingSignal
```

**延迟执行的原因**：
- 交叉信号基于收盘价计算，收盘后才能确认
- 严格遵循时序原则，禁止使用未来数据
- 实际交易中，信号确认后只能在下一交易日执行

### 3.3 信号处理流程

```
输入: KLine_T (第T天的K线数据)

Step 1: 检查是否有待执行信号
  如果存在 PendingSignal 且 Date != T日：
    a. 如果当前有持仓，先平仓（平仓价 = T日开盘价）
    b. 按信号方向开仓（开仓价 = T日开盘价）
    c. 清除 PendingSignal

Step 2: 更新均线状态
  使用T日收盘价更新短期和长期均线

Step 3: 检测交叉信号
  如果均线就绪：
    检测金叉 → 设置 PendingSignal{Buy, T日}
    检测死叉 → 设置 PendingSignal{Sell, T日}
```

### 3.4 交易示例

**示例1: 金叉做多**
```
日期       收盘价   短期MA(5)  长期MA(20)  操作
--------------------------------------------------------
Day 19     3800     -          -           均线未就绪
Day 20     3850     3810       3820        均线就绪，无交叉
Day 21     3900     3830       3825        无交叉
Day 22     3950     3870       3835        金叉! 记录PendingSignal{Buy}
Day 23     3920     3894       3843        开多@3920(开盘价)
```

**示例2: 死叉做空**
```
日期       收盘价   短期MA(5)  长期MA(20)  操作
--------------------------------------------------------
Day 50     4000     3980       3990        无交叉
Day 51     3950     3960       3988        无交叉
Day 52     3900     3940       3985        死叉! 记录PendingSignal{Sell}
Day 53     3880     3916       3980        平多@3880, 开空@3880
```

**示例3: 连续交叉**
```
日期       收盘价   短期MA(5)  长期MA(20)  操作
--------------------------------------------------------
Day 60     4050     4010       4000        金叉! PendingSignal{Buy}
Day 61     4100     4040       4008        开多@4100
Day 70     3850     3900       3950        死叉! PendingSignal{Sell}
Day 71     3800     3870       3940        平多@3800, 开空@3800
```

## 4. 杠杆计算

### 4.1 杠杆使用

MA策略的杠杆系数直接作为持仓杠杆使用，不进行动态计算（与阴阳线策略不同）：

```
持仓杠杆 = leverage 参数值
```

### 4.2 与阴阳线策略的区别

阴阳线策略根据开仓价与止损价的距离动态计算杠杆，而MA策略没有基于形态的止损价概念，因此直接使用参数指定的杠杆值。

## 5. 移仓换月

### 5.1 移仓触发

与阴阳线策略相同，当检测到主力合约切换时触发移仓换月流程：
- T日收盘后：检测主力合约切换
- T+1日开盘：执行移仓换月

### 5.2 移仓流程

```
1. 使用旧合约开盘价平仓
2. 通过模拟交易确定新合约方向
3. 使用新合约开盘价开仓
```

### 5.3 新合约方向确定

**模拟交易法**：
1. 获取新合约从上市首日到移仓前一日的所有历史K线
2. 使用相同参数的MA策略模拟在新合约上连续交易
3. 以模拟结果确定的持仓方向作为新合约开仓方向

**特点**：
- 新合约方向独立于旧合约
- 方向可能与旧合约相反
- 符合策略逻辑的一致性

### 5.4 多合约状态管理

- 每个合约维护独立的均线状态（StateManager）
- 所有合约状态每日同步更新
- 移仓换月时新合约的均线状态是完整的

## 6. 时序性要求

### 6.1 核心原则

**严格遵循时序，禁止使用未来函数**

### 6.2 时序关键点

| 时间点 | 操作 | 使用的数据 |
|--------|------|-----------|
| T日收盘后 | 计算均线、检测交叉 | T日及之前的收盘价 |
| T日收盘后 | 记录PendingSignal | T日的交叉判断结果 |
| T+1日开盘 | 执行交易 | T+1日开盘价 |
| T+1日收盘后 | 更新均线 | T+1日收盘价 |

### 6.3 禁止事项

- 禁止在交叉信号当天执行交易（必须延迟到下一根K线）
- 禁止使用T+1日的收盘价影响T日的交叉判断
- 禁止在移仓检测日立即执行移仓

## 7. 策略优势与风险

### 7.1 优势
- 逻辑简单清晰，易于理解和实现
- 能有效捕捉中长期趋势
- 信号客观明确，无主观判断
- 延迟执行避免未来函数问题

### 7.2 风险
- 震荡市容易频繁交叉，产生大量虚假信号
- 信号滞后，入场和出场均晚于趋势起点和终点
- 均线周期选择对策略表现影响显著
- 短周期参数可能导致过度交易

### 7.3 适用场景
- 日线级别的趋势行情
- 流动性好的主流品种
- 中长期投资周期
- 趋势明显的市场环境

## 8. 参数优化建议

### 8.1 常见参数组合

| 组合 | 短期 | 长期 | 特点 |
|------|------|------|------|
| 短线 | 5 | 20 | 信号频繁，适合短线趋势 |
| 中线 | 10 | 30 | 信号适中，平衡灵敏度和稳定性 |
| 长线 | 20 | 60 | 信号稀少，适合大级别趋势 |
| 经典 | 5 | 10 | 非常灵敏，适合快速切换的市场 |

### 8.2 参数选择原则

- 短期周期必须小于长期周期
- 周期差越大，信号越少但越可靠
- 周期差越小，信号越多但虚假信号也越多
- 应根据品种特性和市场环境选择合适的参数
